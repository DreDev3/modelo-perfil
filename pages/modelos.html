Modelos de páginas criadas

Cópia do Instagram

Uma cópia feita a partir da página web do Instagram, criada no intuito de demonstrar algumas das minhas habilidades em HTML, CSS e alguns efeitos criados com Javascript, como o efeito nos story's que podem ser passados ao clicar nos botões posicionados dos lados direito e esquerdo da caixa na qual se encontram os story's. Também há as configurações de um post que, ao clicar nos [...] nas publicações, aparecerá sobre a tela.

Modelo de site demonstrativo Mustang

Um modelo de página projetado para demonstrar meus conhecimentos nas linguagens HTML e CSS, na estruturação de uma página, assim como um breve código na linguagem de programação Javascript para criar o relógio no topo da página, e o uso de uma API para envio de e-mail direto pelo formulário da página.
